## What you can do

Connect this agent connector to let your agent:

- **List workflows, view, subscriptions** — List all workflows in the project
- **Get workflows, view, surveys** — Get a specific workflow by ID
- **Update view, feature flag, survey** — Update an existing data warehouse saved query (view)
- **Unmaterialize view** — Undo materialization for a saved query
- **Run view, evaluation** — Get the 5 most recent materialization run statuses for a saved query
- **Materialize view** — Enable materialization for a saved query
