## What you can do

Connect this agent connector to let your agent:

- **List workspaces, insights public, forms public** — List the workspaces the caller can see, with id, name, form_count, type (private/shared/custom), and account_id
- **Discover insights public** — Return the schema of analytics data available for a given scope
- **Get forms public, contacts public** — Retrieve a form
- **Delete forms public, contacts public** — Delete/remove a form based on its ID
- **Create forms public, contacts public** — Create a new Typeform form
- **Update contacts public** — Update an existing form property mapping (sync config)
