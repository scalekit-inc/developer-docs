## What you can do

Connect this agent connector to let your agent:

- **Files write siteui, write, edit siteui** — Write (create or overwrite) multiple source files in a @siteui shared package
- **File write, edit** — Write (create or overwrite) a single source file in a vibe section's React app
- **Whoami records** — Get the authenticated Onepage account identity (email, name, language, role)
- **Media upload** — Upload a media file (image, video, document) to the site's media library
- **Update site settings, page settings, crm form** — Patch site-level settings
- **Page unpublish, publish, archive** — Unpublish a page
