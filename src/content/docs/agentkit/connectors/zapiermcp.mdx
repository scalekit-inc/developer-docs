## What you can do

Connect this agent connector to let your agent:

- **Update zapier skill** — Update an existing Zapier Skill's description or content by name
- **Send feedback** — Send feedback about your Zapier MCP experience to the Zapier team
- **List zapier skills, enabled zapier actions** — List all saved Zapier Skills with their names and descriptions
- **Get zapier skill, configuration url** — Fetch the full markdown content of a Zapier Skill by name
- **Execute zapier write action, zapier read action** — Execute a write or create action in a connected app
- **Action enable zapier, disable zapier** — Enable an app's actions on this MCP server
