## What you can do

Connect this agent connector to let your agent:

- **Create attachment, attachment from upload, issue label** — Deprecated fallback for tiny files only
- **Delete attachment, comment, status update** — Delete an attachment by ID
- **Images extract** — Extract and fetch images from markdown content
- **Get attachment, diff, diff threads** — Retrieve an attachment's content by ID
- **List comments, cycles, diffs** — List comments on a Linear issue, project, initiative, document, or project milestone
- **Upload prepare attachment** — Prepare a direct Linear file upload for an existing issue
