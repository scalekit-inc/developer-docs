## What you can do

Connect this agent connector to let your agent:

- **Template upload, download** — Upload and store a reusable Carbone template for use with render_document
- **Update template metadata** — Update a stored template's name, comment, category, tags, or deployment timestamps
- **Document render, convert** — Generate a document by merging a Carbone template with JSON data, optionally converting the output format
- **List templates, tags, categories** — List stored Carbone templates with optional filtering by category, tag, ID, or search query
- **Get capabilities, api status** — Return a summary of all Carbone capabilities including supported formats, features, and usage examples
- **Delete template** — Soft-delete a stored Carbone template by its template ID
