## What you can do

Connect this agent connector to let your agent:

- **List organizations** — Discover all LogRocket organizations the authenticated user can access
- **List projects** — Identify accessible projects within a LogRocket organization
- **Use LogRocket** — Query sessions, metrics, and issues using natural language to investigate bugs, analyze behavior, and detect regressions

Use the exact tool names from the **Tool list** below when you call `execute_tool`. If you're not sure which name to use, list the organizations available for the current user first.
