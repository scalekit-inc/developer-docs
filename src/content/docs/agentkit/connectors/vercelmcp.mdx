## What you can do

Connect this agent connector to let your agent:

- **List projects and deployments** — Retrieve all Vercel projects and their deployment history for a team
- **Get deployment details** — Fetch deployment metadata, build logs, and runtime logs for any project
- **Deploy to Vercel** — Trigger a new deployment for a Vercel project directly from your workflow
- **Check domain availability** — Look up whether a domain name is available and its price
- **Manage toolbar threads** — Create, list, and reply to comment threads on Vercel deployments
- **Fetch and search Vercel content** — Retrieve the response body of any Vercel URL or search the Vercel documentation
