## What you can do

Connect this agent connector to let your agent:

- **Update update** — Update the vacation auto-reply settings for the authenticated Gmail account
- **Message trash** — Move a Gmail message to the Trash
- **File share, move, copy** — Share a file or folder in Google Drive by creating a new permission for a user, group, domain, or anyone
- **Query query** — Query Google Drive activity to see who viewed, edited, moved, or shared files
- **Labels modify message** — Add or remove labels on a Gmail message
- **List list** — List matters in Google Vault

## Authentication

This connector uses **Service Account (DWD)** authentication.
