   Full SDK reference: [Node.js](/agentkit/sdks/node/) | [Python](/agentkit/sdks/python/)

2. ### Set your credentials

   Add these to your environment. Find values in **[app.scalekit.com](https://app.scalekit.com)** > **Developers** > **API Credentials**.

   ```bash frame="terminal"
   export SCALEKIT_ENV_URL="https://your-env.scalekit.com"
   export SCALEKIT_CLIENT_ID="..."
   export SCALEKIT_CLIENT_SECRET="..."
   ```

3. ### Set up the connector

   Register your Microsoft Excel credentials with Scalekit so it handles the token lifecycle. You do this once per environment.
