## What you can do

Connect this agent connector to let your agent:

- **Forget records** — Forget stored data by permanently deleting an entire dataset from Cognee memory, including its data items and the knowledge graph built from them
- **Improve records** — Improve stored memory by running Cognee's cognify step over one or more datasets
- **Recall records** — Recall data previously saved to Cognee memory
- **Remember records** — Save data to Cognee memory
