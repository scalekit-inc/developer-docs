## Authentication

This connector uses **OAuth 2.0**. Scalekit acts as the OAuth client: it redirects your user to Monday.com, obtains an access token, and automatically refreshes it before it expires. Your agent code never handles tokens directly: you only pass a `connectionName` and a user `identifier`.

You supply your Monday.com **Connected App** credentials (Client ID + Secret) once per environment in the Scalekit dashboard.
