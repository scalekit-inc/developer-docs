## What you can do

Connect this agent connector to let your agent:

- **File upload, download, checkout** — Create an upload session for uploading a file to a SharePoint document library
- **Update site, list item, list field** — Update the display name or description of an existing SharePoint site
- **Document unfollow, follow** — Stop following a SharePoint document or OneDrive file
- **Webhook subscribe** — Create a webhook subscription to receive change notifications for a SharePoint list or site resource
- **Search records** — Search across SharePoint sites, lists, drive items, and list items using the Microsoft Search API
- **Item restore recycled, recycle** — Restore a previously recycled (soft-deleted) item in a SharePoint document library
