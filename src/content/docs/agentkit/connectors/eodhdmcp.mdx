## What you can do

Connect this agent connector to let your agent:

- **Screener stock** — Screen and filter stocks by fundamental and technical criteria
- **Id retrieve description by** — Retrieve built-in EODHD API documentation by numeric type and id
- **Ticker resolve** — Resolve a company name, partial ticker, or ISIN to SYMBOL.EXCHANGE format (and ISIN)
- **List mp indices** — [Marketplace] List all available S&P and Dow Jones indices with end-of-day details
- **Components mp index** — [Marketplace] Get constituent stocks of a specific S&P or Dow Jones index, including
historical component changes for major indices
- **Insights mp illio risk, mp illio performance** — [Illio] Retrieve portfolio-level risk attributes for a major US index
