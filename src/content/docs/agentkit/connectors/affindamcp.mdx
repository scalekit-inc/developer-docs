## What you can do

Connect this agent connector to let your agent:

- **Processing wait for document** — Block until every document in a workspace has finished processing
- **Update workspace, validation rule, organization** — Update one or more settings on an existing workspace
- **Connection test** — Verify a service connection's credentials are still valid
- **Secret set integration** — Create or update a secret on an integration
- **Run integration** — Execute an integration against one document as a test run
- **Version revert integration, deploy integration** — Roll an integration back to a previous version and redeploy it
