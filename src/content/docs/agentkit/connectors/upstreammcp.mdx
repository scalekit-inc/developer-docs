## What you can do

Connect this agent connector to let your agent:

- **Update settings, rule, inbox split** — Update user settings like auto-draft configuration, theme, and notification preferences
- **Threads trash, star** — Move threads to trash or restore them
- **Thread snooze, save draft, reply to** — Snooze a thread until a specific date/time
- **Search inbox** — Search across all inbox threads by keyword or phrase
- **Reply save draft** — Save a draft reply in an existing thread without sending it
- **Read thread comments, thread, mark** — Read internal team comments on a thread (not visible to external recipients)
