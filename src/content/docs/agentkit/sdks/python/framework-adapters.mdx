Pre-built integrations for LangChain (and Google ADK via `actions.google`). Use these when your agent needs native tool objects. For a framework-agnostic path, prefer Virtual MCP Servers.

Install LangChain support separately:

```bash
pip install langchain
```

`scalekit_client.actions.langchain` is a lazy helper on `ActionClient` that converts Scalekit scoped tools into LangChain `StructuredTool` instances.
