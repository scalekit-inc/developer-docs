      (function() {
        // Function to initialize the copy functionality
        function initHeadingCopy() {
          const headings = document.querySelectorAll('.sl-markdown-content h2, .sl-markdown-content h3, .sl-markdown-content h4');

          headings.forEach(heading => {
            // Skip if already initialized
            if (heading.hasAttribute('data-copy-initialized')) return;
            heading.setAttribute('data-copy-initialized', 'true');

            heading.addEventListener('click', async () => {
              const headingText = heading.textContent.replace(/\s📋$/, '').trim();

              try {
                await navigator.clipboard.writeText(headingText);

                // Visual feedback
                const originalText = heading.textContent;
                heading.textContent = heading.textContent.replace(/\s📋$/, '') + ' ✓';

                setTimeout(() => {
                  heading.textContent = originalText;
                }, 2000);
              } catch (err) {
                console.error('Failed to copy heading:', err);
              }
            });
          });
        }

        // Initialize on DOMContentLoaded
        document.addEventListener('DOMContentLoaded', initHeadingCopy);

        // Also initialize now in case the DOM is already loaded
        if (document.readyState === 'complete' || document.readyState === 'interactive') {
          setTimeout(initHeadingCopy, 1);
        }

        // For Astro's client-side navigation
        document.addEventListener('astro:page-load', initHeadingCopy);

## Tool list

## `apollo_contacts_search`

Search for contacts in Apollo using filters such as name, title, company, location, and more.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `contact_stage_ids` | `array<string>` | No | Filter by contact stage IDs |
| `email_status` | string | No | Filter by email verification status (e.g., `verified`, `unverified`) |
| `organization_ids` | `array<string>` | No | Filter contacts by organization IDs |
| `page` | integer | No | Page number for pagination |
| `per_page` | integer | No | Number of results per page (max 100) |
| `person_locations` | `array<string>` | No | Filter by contact location |
| `person_titles` | `array<string>` | No | Filter by job title keywords |
| `q_keywords` | string | No | Full-text keyword search |
| `sort_ascending` | boolean | No | Sort results in ascending order |
| `sort_by_field` | string | No | Field to sort results by |

## `apollo_contact_get`

Retrieve details of a specific contact from Apollo by contact ID.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `contact_id` | string | Yes | ID of the contact to retrieve |

## `apollo_contact_create`

Create a new contact in Apollo.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `account_id` | string | No | ID of the account to associate this contact with |
| `email` | string | No | Email address of the contact |
| `first_name` | string | No | First name of the contact |
| `label_names` | `array<string>` | No | Labels to apply to the contact |
| `last_name` | string | Yes | Last name of the contact |
| `organization_name` | string | No | Name of the contact's organization |
| `phone_numbers` | `array<object>` | No | Array of phone number objects |
| `title` | string | No | Job title of the contact |

## `apollo_contact_update`

Update an existing contact in Apollo by contact ID.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `account_id` | string | No | ID of the account to associate this contact with |
| `contact_id` | string | Yes | ID of the contact to update |
| `email` | string | No | Updated email address |
| `first_name` | string | No | Updated first name |
| `label_names` | `array<string>` | No | Labels to apply to the contact |
| `last_name` | string | No | Updated last name |
| `organization_name` | string | No | Updated organization name |
| `title` | string | No | Updated job title |

## `apollo_accounts_search`

Search for accounts (organizations) in Apollo using filters such as industry, employee count, and location.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `account_stage_ids` | `array<string>` | No | Filter by account stage IDs |
| `industries` | `array<string>` | No | Filter by industry keywords |
| `maximum_employees` | integer | No | Maximum number of employees |
| `minimum_employees` | integer | No | Minimum number of employees |
| `page` | integer | No | Page number for pagination |
| `per_page` | integer | No | Number of results per page (max 100) |
| `q_keywords` | string | No | Full-text keyword search |
| `sort_ascending` | boolean | No | Sort results in ascending order |
| `sort_by_field` | string | No | Field to sort results by |

## `apollo_account_get`

Retrieve details of a specific account from Apollo by account ID.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `account_id` | string | Yes | ID of the account to retrieve |

## `apollo_people_enrich`

Enrich a person's profile with additional data from Apollo's database using their email or name and company.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `email` | string | No | Email address to enrich (recommended) |
| `first_name` | string | No | First name of the person |
| `last_name` | string | No | Last name of the person |
| `organization_name` | string | No | Organization name for the person |

## `apollo_organizations_enrich`

Enrich an organization's profile with additional data from Apollo's database using domain or name.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `domain` | string | No | Domain of the organization (e.g., `example.com`) |
| `name` | string | No | Name of the organization |

## `apollo_sequences_list`

List all sequences (email outreach campaigns) in Apollo.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `page` | integer | No | Page number for pagination |
| `per_page` | integer | No | Number of results per page |

## `apollo_sequence_contacts_add`

Add contacts to an Apollo sequence to enroll them in an email outreach campaign.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `contact_ids` | `array<string>` | Yes | Array of contact IDs to add to the sequence |
| `sequence_id` | string | Yes | ID of the sequence to add contacts to |
| `user_id` | string | No | ID of the user sending the sequence |
