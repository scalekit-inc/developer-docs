## What you can do

Connect this agent connector to let your agent:

- **Terminate cluster** — Terminate a Databricks cluster by cluster ID
- **List job runs, jobs, sql warehouses** — List all job runs in the Databricks workspace, optionally filtered by job ID
- **Get scim me, cluster, job** — Retrieve information about the currently authenticated service principal in the Databricks workspace
- **Execute sql statement** — Execute a SQL statement on a Databricks SQL warehouse and return the results
- **Start cluster** — Start a terminated Databricks cluster by cluster ID
- **Run job** — Trigger an immediate run of a Databricks job by job ID

## Authentication

This connector uses **Service Principal (OAuth 2.0)** authentication.
