## Tool list

## `google_looker_dashboards_all`

Retrieve a list of all dashboards the authenticated user has access to in Looker.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |

## `google_looker_dashboard_get`

Retrieve details and configuration of a specific Looker dashboard by ID.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `dashboard_id` | string | Yes | ID of the dashboard to retrieve |
| `fields` | string | No | Comma-separated list of fields to include in the response |

## `google_looker_dashboard_run`

Run a Looker dashboard and return its tile results.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `dashboard_id` | string | Yes | ID of the dashboard to run |
| `filters` | `object` | No | Dashboard filter values to apply |

## `google_looker_looks_all`

Retrieve a list of all saved Looks (saved queries) the authenticated user has access to.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |

## `google_looker_look_get`

Retrieve details of a specific Look by ID.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |
| `look_id` | integer | Yes | ID of the Look to retrieve |

## `google_looker_look_run`

Run a Look and return its query results in the specified format.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `apply_formatting` | boolean | No | Apply model-specified formatting to results |
| `cache` | boolean | No | Use cached results if available |
| `limit` | integer | No | Maximum number of rows to return |
| `look_id` | integer | Yes | ID of the Look to run |
| `result_format` | string | Yes | Output format: `json`, `json_detail`, `csv`, `txt`, `html`, `md`, `xlsx`, `sql` |

## `google_looker_query_create_and_run`

Create and run a Looker query inline, returning results immediately.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | `array<string>` | Yes | List of field names to include in the query |
| `filters` | `object` | No | Filter conditions as key-value pairs |
| `limit` | string | No | Maximum number of rows to return |
| `model` | string | Yes | Name of the LookML model |
| `result_format` | string | Yes | Output format: `json`, `json_detail`, `csv` |
| `sorts` | `array<string>` | No | Sort order for results |
| `view` | string | Yes | Name of the LookML view (Explore) |

## `google_looker_folders_all`

Retrieve a list of all folders in Looker.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |

## `google_looker_folder_get`

Retrieve details of a specific folder by ID, including its contents.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |
| `folder_id` | string | Yes | ID of the folder to retrieve |

## `google_looker_explores_all`

List all Explores available in a given LookML model.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |
| `lookml_model_name` | string | Yes | Name of the LookML model |

## `google_looker_users_all`

Retrieve a list of all users in the Looker instance.

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `fields` | string | No | Comma-separated list of fields to include in the response |
| `page` | integer | No | Page number for pagination |
| `per_page` | integer | No | Number of results per page |
