## What you can do

Connect this agent connector to let your agent:

- **Create create** — Create a note on a person, organization, or opportunity in Affinity
- **Get get** — Retrieve full details of a deal or opportunity in Affinity including current stage, owner, associated persons and organizations, custom field values, and list membership
- **List list** — List pipeline opportunities in Affinity with optional filters by list ID, owner, or stage
- **Search search** — Search for people in the Affinity network by name, email, or relationship strength
- **Update update** — Update an existing deal or opportunity in Affinity

## Authentication

This connector uses **Bearer Token** authentication. Scalekit securely stores the token and injects it into API requests on behalf of your users. Your agent code never handles tokens directly — you only pass a `connectionName` and a user `identifier`.
