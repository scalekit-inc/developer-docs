## User Authentication Events

### `user.signup`

This webhook is triggered when a new user signs up. The event type is `user.signup`

```json title="user.signup"
{
  "spec_version": "1",
  "id": "evt_99772985350750722",
  "object": "OrgMembershipEvent",
  "environment_id": "env_74418471961625391",
  "occurred_at": "2025-11-19T07:15:33.507250461Z",
  "type": "user.signup",
  "data": {
    "organization": {
      "id": "org_99772985317065218",
      "environment_id": "env_74418471961625391",
      "region": "us",
      "deleted_at": null
    },
    "user": {
      "id": "usr_99772985317130754",
      "email": "sara.namas@example.com",
      "environment_id": "env_74418471961625391",
      "create_time": "2025-11-19T07:15:33Z",
      "update_time": "2025-11-19T07:15:33Z",
      "user_profile": {
        "id": "usp_99772985317196290",
        "name": "Sara Namas",
        "given_name": "Sara",
        "family_name": "Namas",
        "email_verified": true,
        "picture": "https://example.com/profile-picture.jpg"
      },
      "membership": {
        "organization_id": "org_99772985317065218",
        "membership_status": "ACTIVE",
        "provisioning_method": "org_creator",
        "created_at": "2025-11-19T07:15:33Z",
        "accepted_at": "2025-11-19T07:15:33Z",
        "roles": [
          {
            "id": "role_75142006447276105"
          }
        ]
      }
    }
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `organization` | object | Organization details associated with the signup |
| `organization.id` | string | Unique identifier for the organization |
| `organization.environment_id` | string | Identifier for the environment |
| `organization.region` | string | The region where the organization is hosted |
| `organization.deleted_at` | string \| null | Timestamp of when the organization was deleted, null if active |
| `user` | object | User details for the signed-up user |
| `user.id` | string | Unique identifier for the user |
| `user.email` | string | Email address of the user |
| `user.environment_id` | string | Identifier for the environment |
| `user.create_time` | string | Timestamp of when the user was created |
| `user.update_time` | string | Timestamp of when the user was last updated |
| `user.user_profile` | object | User profile information |
| `user.user_profile.id` | string | Unique identifier for the user profile |
| `user.user_profile.name` | string | Full name of the user |
| `user.user_profile.given_name` | string | Given name of the user |
| `user.user_profile.family_name` | string | Family name of the user |
| `user.user_profile.email_verified` | boolean | Indicates if the user's email is verified |
| `user.user_profile.picture` | string | URL of the user's profile picture |
| `user.membership` | object | Organization membership details |
| `user.membership.organization_id` | string | Identifier for the organization |
| `user.membership.membership_status` | string | Status of the membership (ACTIVE, PENDING, etc.) |
| `user.membership.provisioning_method` | string | Method used for provisioning (org_creator, JIT, etc.) |
| `user.membership.created_at` | string | Timestamp of when the membership was created |
| `user.membership.accepted_at` | string | Timestamp of when the membership was accepted |
| `user.membership.roles` | array | Array of roles assigned to the user |

| Field | Type | Description |
|-------|------|-------------|

| `user_session.identity` | string | User identifier associated with the session |
| `user_session.environment_id` | string | Identifier for the environment |

| `user_session.updated_at` | string | Timestamp of when the session was last updated |
| `user_session.deleted` | string \| null | Timestamp of when the session was deleted, null if active |
| `user_session.details` | object | Additional session details including auth request and connection information |
