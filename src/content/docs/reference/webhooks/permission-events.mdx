----

## Permission events

### `permission.created`

This webhook is triggered when a new permission is created. The event type is `permission.created`

```json title="permission.created"
{
  "spec_version": "1",
  "id": "evt_98932087654321098",
  "object": "Permission",
  "environment_id": "env_74418471961625391",
  "occurred_at": "2025-11-13T12:05:30.123456789Z",
  "type": "permission.created",
  "data": {
    "id": "perm_98932087654321098",
    "name": "read:reports",
    "display_name": "Read Reports",
    "description": "Permission to view and read reports"
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique identifier for the permission |
| `name` | string | Unique name identifier for the permission |
| `display_name` | string | Human-readable display name for the permission |
| `description` | string | Description of what the permission allows |

### `permission.updated`

This webhook is triggered when a permission is updated. The event type is `permission.updated`

```json title="permission.updated"
{
  "spec_version": "1",
  "id": "evt_98932088765432109",
  "object": "Permission",
  "environment_id": "env_74418471961625391",
  "occurred_at": "2025-11-13T12:10:45.234567890Z",
  "type": "permission.updated",
  "data": {
    "id": "perm_98932087654321098",
    "name": "read:reports",
    "display_name": "View Reports",
    "description": "Permission to view and access all reports"
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique identifier for the permission |
| `name` | string | Unique name identifier for the permission |
| `display_name` | string | Human-readable display name for the permission |
| `description` | string | Description of what the permission allows |

### `permission.deleted`

This webhook is triggered when a permission is deleted. The event type is `permission.deleted`

```json title="permission.deleted"
{
  "spec_version": "1",
  "id": "evt_98932089876543210",
  "object": "Permission",
  "environment_id": "env_74418471961625391",
  "occurred_at": "2025-11-13T12:15:20.345678901Z",
  "type": "permission.deleted",
  "data": {
    "id": "perm_98932087654321098",
    "name": "read:reports",
    "display_name": "View Reports",
    "description": "Permission to view and access all reports"
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique identifier for the deleted permission |
| `name` | string | Unique name identifier for the deleted permission |
| `display_name` | string | Human-readable display name for the deleted permission |
| `description` | string | Description of what the permission allowed |
