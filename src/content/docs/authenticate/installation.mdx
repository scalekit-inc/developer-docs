## Prerequisites

Before you begin, make sure you have:

- A Scalekit account (sign up at [scalekit.com](https://scalekit.com))
- Your application's development environment set up
- Basic knowledge of your application's tech stack

## Installation Steps

### 1. Set up your Scalekit account

Follow the [Setup Scalekit guide](/guides/setup-scalekit) to configure your account and obtain your API credentials.

### 3. Configure your environment

Set up your environment variables with your Scalekit credentials:

```bash
SCALEKIT_ENV_URL=https://your-env.scalekit.com
SCALEKIT_CLIENT_ID=your_client_id
SCALEKIT_CLIENT_SECRET=your_client_secret
```
