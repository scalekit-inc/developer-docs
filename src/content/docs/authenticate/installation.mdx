<Steps>
1. ## Create a Scalekit account

   Scalekit provides dedicated development and production environments. Use them to manage your authentication services for each stage of your app.

   <LinkButton href="https://app.scalekit.com/ws/signup" variant="secondary"  class='sk-secondary'  icon='right-arrow'>Create a Scalekit Account</LinkButton>

2. ## Set up your environments

   After creating your account, you will have two environments:

   ```md showLineNumbers=false wrap title="Environment URLs"
   https://{your-subdomain}.scalekit.dev  (Development)
   https://{your-subdomain}.scalekit.com  (Production)
   ```

   Each environment is isolated. You can map a custom domain (for example, `auth.example.com`). See the [custom domain guide](/guides/custom-domain/).

   View your environment URLs in **Dashboard > Developers > Settings**.

3. ## Get your API credentials

   Scalekit uses the OAuth 2.0 client credentials flow for secure API authentication.

   In **Developers > Settings > API config**, copy these values:

   ```sh showLineNumbers=false wrap title="Environment variables"
   SCALEKIT_ENVIRONMENT_URL=<your-environment-url>
   # Example: https://acme.scalekit.dev or https://auth.acme.com

   SCALEKIT_CLIENT_ID=<app-client-id>
   # Example: skc_1234567890abcdef

   SCALEKIT_CLIENT_SECRET=<app-client-secret>
   # Example: test_abcdef1234567890
   ```

5. ## Verify your setup

   Initialize the SDK, then list organizations to confirm your credentials work.

   <Tabs syncKey="tech-stack">
   <TabItem value="curl" label="cURL">

    ```bash wrap showLineNumbers=false title="Authenticate with client credentials"
    # Get an access token
    curl https://<SCALEKIT_ENVIRONMENT_URL>/oauth/token \
      -X POST \
      -H 'Content-Type: application/x-www-form-urlencoded' \
      -d 'client_id=<SCALEKIT_CLIENT_ID>' \
      -d 'client_secret=<SCALEKIT_CLIENT_SECRET>' \
      -d 'grant_type=client_credentials'
    ```

    This returns an access token:

    ```json wrap showLineNumbers=false
    {
      "access_token": "eyJhbGciOiJSUzI1NiIsImInR5cCI6IkpXVCJ9...",
      "token_type": "Bearer",
      "expires_in": 86399,
      "scope": "openid"
    }
    ```

    **Step 2: Use the token to list organizations**

    ```bash wrap showLineNumbers=false title="List organizations"
    curl -L 'https://<SCALEKIT_ENVIRONMENT_URL>/api/v1/organizations?page_size=5' \
      -H 'Authorization: Bearer <ACCESS_TOKEN>'
    ```
