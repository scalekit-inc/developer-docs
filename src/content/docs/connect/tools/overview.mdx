Tools are the core building blocks of Agent Connect that enable you to perform actions across different third-party providers through a unified interface. Each tool represents a specific operation that can be executed on behalf of a connected user.

## What are tools?

Tools in Agent Connect are standardized functions that:

- **Abstract API complexity**: Hide provider-specific API details behind simple interfaces
- **Ensure consistency**: Provide uniform input/output formats across all providers
- **Handle authentication**: Automatically use connected account tokens
- **Manage errors**: Provide consistent error handling and responses
- **Support batching**: Allow multiple operations to be executed efficiently

## Tool structure

Every tool in Agent Connect follows a consistent structure:

### Tool definition

```json
{
  "name": "send_email",
  "display_name": "Send Email",
  "description": "Send an email message to one or more recipients",
  "provider": "gmail",
  "category": "communication",
  "input_schema": {
    "type": "object",
    "properties": {
      "to": {
        "type": "array",
        "items": {"type": "string", "format": "email"},
        "description": "Email addresses of recipients"
      },
      "subject": {
        "type": "string",
        "description": "Email subject line"
      },
      "body": {
        "type": "string",
        "description": "Email body content"
      }
    },
    "required": ["to", "subject", "body"]
  },
  "output_schema": {
    "type": "object",
    "properties": {
      "message_id": {
        "type": "string",
        "description": "Unique identifier for the sent message"
      },
      "status": {
        "type": "string",
        "enum": ["sent", "queued", "failed"],
        "description": "Status of the email sending operation"
      }
    }
  }
}
```

### Tool execution

```javascript
// Execute the tool with parameters
const result = await agentConnect.tools.execute({
  connected_account_id: 'account_123',
  tool: 'send_email',
  parameters: {
    to: ['recipient@example.com'],
    subject: 'Hello from Agent Connect',
    body: 'This is a test email'
  }
});

// Response follows the output schema
console.log(result.message_id); // "msg_456"
console.log(result.status);     // "sent"
```

## Common tools by provider

### Gmail tools

**Communication tools:**
- `send_email` - Send email messages
- `fetch_emails` - Retrieve email messages
- `create_draft` - Create email drafts
- `reply_to_email` - Reply to existing emails
- `forward_email` - Forward email messages
- `search_emails` - Search through emails
- `mark_as_read` - Mark emails as read/unread
- `delete_email` - Delete email messages

**Example usage:**
```javascript
// Send an email with attachment
const result = await agentConnect.tools.execute({
  connected_account_id: 'gmail_account',
  tool: 'send_email',
  parameters: {
    to: ['customer@example.com'],
    subject: 'Your invoice is ready',
    body: 'Please find your invoice attached.',
    attachments: [
      {
        filename: 'invoice.pdf',
        content: base64EncodedPDF,
        mime_type: 'application/pdf'
      }
    ]
  }
});
```

**Scheduling tools:**
- `create_event` - Create calendar events
- `fetch_events` - Retrieve calendar events
- `update_event` - Update existing events

## Tool parameters and validation

### Parameter types

Tools support various parameter types:

- **String**: Text values, emails, URLs
- **Number**: Integers and floating-point numbers
- **Boolean**: True/false values
- **Array**: Lists of values
- **Object**: Complex nested structures
- **File**: File uploads and attachments

### Parameter validation

Agent Connect automatically validates parameters:

```javascript
// Invalid parameters will be rejected
try {
  await agentConnect.tools.execute({
    connected_account_id: 'account_123',
    tool: 'send_email',
    parameters: {
      to: 'invalid-email', // Should be array
      subject: '',          // Required field is empty
      body: 'Test'
    }
  });
} catch (error) {
  console.log(error.code); // 'INVALID_PARAMETERS'
  console.log(error.validation_errors); // Detailed validation errors
}
```

### Optional and required parameters

Tools clearly define which parameters are required:

```javascript
// Required parameters
{
  "required": ["to", "subject", "body"],
  "optional": ["cc", "bcc", "attachments", "html_body"]
}

// Usage with optional parameters
const result = await agentConnect.tools.execute({
  connected_account_id: 'account_123',
  tool: 'send_email',
  parameters: {
    to: ['recipient@example.com'],
    subject: 'Hello',
    body: 'Plain text body',
    html_body: '<p>HTML body</p>',  // Optional
    cc: ['cc@example.com'],         // Optional
    attachments: [...]              // Optional
  }
});
```

## Tool execution patterns

### Synchronous execution

Most tools execute synchronously and return results immediately:

```javascript
const result = await agentConnect.tools.execute({
  connected_account_id: 'account_123',
  tool: 'send_email',
  parameters: { /* ... */ }
});
// Result is available immediately
```

### Asynchronous execution

Some long-running tools support asynchronous execution:

```javascript
const execution = await agentConnect.tools.executeAsync({
  connected_account_id: 'account_123',
  tool: 'bulk_import_contacts',
  parameters: { /* ... */ }
});

// Poll for completion
const result = await agentConnect.tools.getExecution(execution.id);
```

### Batch execution

Execute multiple tools efficiently:

```javascript
const results = await agentConnect.tools.executeBatch([
  {
    connected_account_id: 'gmail_account',
    tool: 'send_email',
    parameters: { /* ... */ }
  },
  {
    connected_account_id: 'slack_account',
    tool: 'send_message',
    parameters: { /* ... */ }
  }
]);
```

## Tool versioning and compatibility

### Version management

Tools are versioned to ensure compatibility:

```javascript
// Use specific tool version
const result = await agentConnect.tools.execute({
  connected_account_id: 'account_123',
  tool: 'send_email',
  version: '1.2.0',
  parameters: { /* ... */ }
});

// Use latest version (default)
const result = await agentConnect.tools.execute({
  connected_account_id: 'account_123',
  tool: 'send_email',
  parameters: { /* ... */ }
});
```

### Deprecation handling

Agent Connect handles tool deprecation gracefully:

```javascript
// Deprecated tools show warnings
const result = await agentConnect.tools.execute({
  connected_account_id: 'account_123',
  tool: 'legacy_send_email',
  parameters: { /* ... */ }
});

// Check for deprecation warnings
if (result.warnings) {
  console.log('Tool is deprecated:', result.warnings);
}
```

## Best practices

### Parameter optimization

- **Handle optionals**: Use optional parameters to enhance functionality

### Error handling

- **Implement retry logic**: Handle transient failures gracefully

- **Monitor rate limits**: Track usage to avoid limit exceeded errors
- **Log errors**: Maintain detailed error logs for debugging

### Performance optimization

- **Cache results**: Cache frequently accessed data
- **Use batch operations**: Reduce API calls through batching
- **Optimize parameters**: Request only necessary data
- **Monitor performance**: Track execution times and optimize

## Tool development and customization

### Custom tool definitions

Create custom tools for specific use cases:

```javascript
// Define a custom tool
const customTool = {
  name: 'send_weekly_report',
  provider: 'gmail',
  category: 'communication',
  description: 'Send weekly project report via email',
  input_schema: {
    type: 'object',
    properties: {
      project_id: { type: 'string' },
      recipients: { type: 'array', items: { type: 'string' } }
    },
    required: ['project_id', 'recipients']
  },
  implementation: async (params, context) => {
    // Custom tool logic
    const report = await generateWeeklyReport(params.project_id);
    return await context.tools.execute({
      tool: 'send_email',
      parameters: {
        to: params.recipients,
        subject: `Weekly Report - ${params.project_id}`,
        body: report
      }
    });
  }
};
```

### Tool composition

Combine multiple tools into workflows:

```javascript
// Compose tools for complex workflows
const createProjectWorkflow = async (projectData) => {
  // Create Jira project
  const project = await agentConnect.tools.execute({
    connected_account_id: 'jira_account',
    tool: 'create_project',
    parameters: projectData
  });

  // Create Slack channel
  const channel = await agentConnect.tools.execute({
    connected_account_id: 'slack_account',
    tool: 'create_channel',
    parameters: {
      name: `${projectData.key.toLowerCase()}-team`,
      topic: `Discussion for ${projectData.name}`
    }
  });

  // Send notification email
  await agentConnect.tools.execute({
    connected_account_id: 'gmail_account',
    tool: 'send_email',
    parameters: {
      to: projectData.team_members,
      subject: `New project created: ${projectData.name}`,
      body: `Project ${projectData.name} has been created. Join the discussion in ${channel.name}.`
    }
  });

  return { project, channel };
};
```
