### get_user_sessions
<div class="sdk-method-section">
  <CB.ClassBrowser name="Sessions" type="client" language="python" source="https://github.com/scalekit-inc/scalekit-sdk-python/blob/main/scalekit/sessions.py">
    <CB.ClassMethod name="get_user_sessions" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Method to get all session details for a user with pagination and filtering support

      <CB.ClassMethodInput name="user_id" type="str">
        User ID to get all session details for
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="page_size" type="int">
        Optional. Number of sessions to return per page
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="page_token" type="str">
        Optional. Token for pagination to get next/previous page
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="filter" type="UserSessionFilter">
        Optional. Filter to apply to sessions (status, time range)
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="UserSessionDetails">
        User Session Details with pagination info
      </CB.ClassMethodOutput>

```python wrap showLineNumbers=false
response = scalekit_client.sessions.get_user_sessions(
    'usr_123456',
    page_size=50
)

for session in response[0].sessions:
    print(f'Session: {session.id}')
```

    </CB.ClassMethod>
  </CB.ClassBrowser>
</div>
