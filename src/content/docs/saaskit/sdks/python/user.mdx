Use `scalekit_client.users` to provision people, manage organization memberships, and assign roles.

Common flows: invite or create a user, attach them to an organization, update profile fields, and look up users by Scalekit ID or your external ID.

      Method to create a new user and add them to an organization

      <CB.ClassMethodInput name="organization_id" type="str">

      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="user" type="CreateUser">

      </CB.ClassMethodInput>

      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="CreateUserAndMembershipResponse">
        Create User And Membership Response
      </CB.ClassMethodOutput>

```python wrap showLineNumbers=false

user.email = "john.doe@example.com"
user.given_name = "John"
user.family_name = "Doe"

response = scalekit_client.users.create_user_and_membership(
    'org_123456',
    user,
    send_invitation_email=True
)
```
