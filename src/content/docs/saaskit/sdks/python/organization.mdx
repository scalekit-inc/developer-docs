Use `scalekit_client.organization` to create tenants, list customers, and update organization settings.

### `list_organizations`

<div class="sdk-method-section">
  <CB.ClassBrowser name="Organization" type="client" language="python" source="https://github.com/scalekit-inc/scalekit-sdk-python/blob/main/scalekit/organization.py">
    <CB.ClassMethod name="list_organizations" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Method to list organizations

      <CB.ClassMethodInput name="page_size" type="Optional[int]">
        Optional. page size for org list fetch (optional, uses server default if not provided)
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="page_token" type="Optional[str]">
        Optional. page token for org list fetch
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="ListOrganizationsResponse">
        list of organizations
      </CB.ClassMethodOutput>

```python wrap showLineNumbers=false
response = scalekit_client.organization.list_organizations(
    page_size=50,
    page_token='next_page_token'
)

for org in response[0].organizations:
    print(f"Organization: {org.display_name}")
```

### `create_organization`

<div class="sdk-method-section">
  <CB.ClassBrowser name="Organization" type="client" language="python" source="https://github.com/scalekit-inc/scalekit-sdk-python/blob/main/scalekit/organization.py">
    <CB.ClassMethod name="create_organization" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Method to create organization based on given data

      <CB.ClassMethodInput name="organization" type="CreateOrganization">
        Create Organization obj with details for org creation
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="CreateOrganizationResponse">
        Create Organization Response
      </CB.ClassMethodOutput>

```python wrap showLineNumbers=false
from scalekit.v1.organizations.organizations_pb2 import CreateOrganization

org = CreateOrganization()
org.display_name = "Acme Corp"
org.external_id = "acme_123"

response = scalekit_client.organization.create_organization(org)
print(f"Created organization: {response[0].organization.id}")
```
