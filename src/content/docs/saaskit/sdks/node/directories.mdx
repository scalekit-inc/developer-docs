Use the `directory` client for SCIM directory sync—listing providers, users, and groups provisioned from an IdP.

Use this after SSO is in place when the customer provisions users via SCIM.

### listDirectories
<div class="sdk-method-section">
  <CB.ClassBrowser name="Directories" type="client" language="typescript" source="https://github.com/scalekit-inc/scalekit-sdk-node/blob/main/src/directory.ts">
    <CB.ClassMethod name="listDirectories" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Retrieves all SCIM directories configured for an organization.

      <CB.ClassMethodInput name="organizationId" type="string">
        Organization ID.
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="ListDirectoriesResponse">
        Directories for the org.
      </CB.ClassMethodOutput>

```typescript wrap showLineNumbers=false
const response = await scalekit.directory.listDirectories('org_123456');

console.log(`Found ${response.directories.length} directories`);
response.directories.forEach(dir => {
  console.log(`- ${dir.provider}: ${dir.enabled ? 'Active' : 'Inactive'}`);
  console.log(`  Users: ${dir.userCount}, Groups: ${dir.groupCount}`);
});
```

    </CB.ClassMethod>
  </CB.ClassBrowser>
</div>

### getDirectory
<div class="sdk-method-section">
  <CB.ClassBrowser name="Directories" type="client" language="typescript" source="https://github.com/scalekit-inc/scalekit-sdk-node/blob/main/src/directory.ts">
    <CB.ClassMethod name="getDirectory" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Retrieves comprehensive details about a specific SCIM directory.

      <CB.ClassMethodInput name="organizationId" type="string">
        Organization ID.
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="directoryId" type="string">
        The directory identifier to retrieve (format: "dir_...")
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="GetDirectoryResponse">
        The directory.
      </CB.ClassMethodOutput>

```typescript wrap showLineNumbers=false
const response = await scalekit.directory.getDirectory(
  'org_123456',
  'dir_abc123'
);

const dir = response.directory;
console.log('Provider:', dir.provider);
console.log('Status:', dir.enabled ? 'Active' : 'Inactive');
console.log('Users:', dir.userCount);
```

    </CB.ClassMethod>
  </CB.ClassBrowser>
</div>

### listDirectoryUsers
<div class="sdk-method-section">
  <CB.ClassBrowser name="Directories" type="client" language="typescript" source="https://github.com/scalekit-inc/scalekit-sdk-node/blob/main/src/directory.ts">
    <CB.ClassMethod name="listDirectoryUsers" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Lists users synchronized from an identity provider via SCIM directory sync.

      <CB.ClassMethodInput name="organizationId" type="string">
        The organization ID
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="directoryId" type="string">
        The directory ID
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="options" type="object" hint="pageSize, pageToken, includeDetail, directoryGroupId, updatedAfter">
        Optional fields: `pageSize`, `pageToken`, `includeDetail`, `directoryGroupId`, `updatedAfter`.
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="ListDirectoryUsersResponse">
        Directory users.
      </CB.ClassMethodOutput>

```typescript wrap showLineNumbers=false
// pageSize: Results per page (default 10, max 100)
// pageToken: Token for the next page
// includeDetail: include detail
// directoryGroupId: directory group id
// updatedAfter: Only resources updated after this time
const response = await scalekit.directory.listDirectoryUsers(
  'org_123456',
  'dir_abc123',
  { pageSize: 50 }
);

console.log(`Found ${response.totalSize} users`);
```

    </CB.ClassMethod>
  </CB.ClassBrowser>
</div>
