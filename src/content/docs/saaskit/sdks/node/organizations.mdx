Manage your customer organizations — create tenants, list customers, and update organization
settings.

Use the `organization` client on `ScalekitClient`:

```ts wrap showLineNumbers=false
scalekit.organization.listOrganization({ pageSize: 20 })
```

        {'Organization that will configure SSO/SCIM.'}
      </CB.ClassMethodInput>

      <CB.ClassMethodOutput type="PortalLink">
        {'One-time portal URL (valid about one minute until opened).'}
      </CB.ClassMethodOutput>
    </CB.ClassMethod>
  </CB.ClassBrowser>
</div>

```ts wrap showLineNumbers=false
app.get('/admin/sso-settings', async (req, res) => {
  const organizationId = req.user.organizationId
  const link = await scalekit.organization.generatePortalLink(organizationId)
  res.redirect(link.location)
})
```
