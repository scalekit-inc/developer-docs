Use the `organization` client on `ScalekitClient` to create tenants, list customers, and update organization settings.

### `createOrganization`

<div class="sdk-method-section">
  <CB.ClassBrowser name="Organizations" type="client" language="typescript" source="https://github.com/scalekit-inc/scalekit-sdk-node/blob/main/src/organization.ts">
    <CB.ClassMethod name="createOrganization" open={true}>
      <CB.ClassModifier type="async" slot="modifiers" />

      Creates a new organization (tenant) in your Scalekit application.

      <CB.ClassMethodInput name="name" type="string">
        Display name for the organization (e.g., "Acme Corporation")
      </CB.ClassMethodInput>
      <CB.ClassMethodInput name="options" type="object" hint="externalId, logoUrl, slug">
        Optional settings.
      </CB.ClassMethodInput>
      <CB.ClassMethodOutput type="CreateOrganizationResponse">
        The created organization.
      </CB.ClassMethodOutput>

```typescript wrap showLineNumbers=false
// externalId: Your system's ID for this org
// logoUrl: Public URL of the org logo
// slug: URL-safe org slug
const org = await scalekit.organization.createOrganization(
  'Acme Corporation',
  { externalId: 'customer_12345' }
);

console.log('Organization ID:', org.organization.id);
```

    </CB.ClassMethod>
  </CB.ClassBrowser>
</div>

  const link = await scalekit.organization.generatePortalLink(organizationId);

  res.redirect(link.location);
});
