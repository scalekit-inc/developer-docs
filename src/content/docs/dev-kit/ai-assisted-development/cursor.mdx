Use Scalekit with Cursor by running the CLI setup, which detects Cursor and configures the auth plugins automatically. Then prompt Cursor to generate the implementation in your existing codebase.

```bash showLineNumbers=false frame="none"
npx @scalekit-inc/cli setup
```

<SupportedAgents hideCommand />

After setup, open Cursor's chat panel with **Cmd+L** (macOS) or **Ctrl+L** (Windows/Linux) and ask your agent to implement Scalekit authentication. See [Build with AI](/dev-kit/build-with-ai/) for product-specific prompts.
