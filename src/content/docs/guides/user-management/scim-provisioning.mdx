SCIM (System for Cross-domain Identity Management) is a protocol that allows you to manage user identities and permissions in your organization. It is a widely adopted standard for managing user identities in enterprise environments.

## Overview

SCIM provisioning is the process of creating, updating, and deleting user accounts in your organization. It is a key part of managing user identities and permissions in your organization.
