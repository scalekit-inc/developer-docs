  <CB.ClassMethod name="webhook">
    Returns a <a href="https://github.com/scalekit-inc/scalekit-sdk-java/blob/main/src/main/java/com/scalekit/webhooks/Webhook.java">Webhook</a> verifier for validating Scalekit webhook payload signatures.

  </CB.ClassMethod>

```java
import java.util.Map;

// Build from the incoming HTTP request
Map<String, String> headers = new java.util.HashMap<>();
headers.put("webhook-id", request.getHeader("webhook-id"));
headers.put("webhook-timestamp", request.getHeader("webhook-timestamp"));
headers.put("webhook-signature", request.getHeader("webhook-signature"));
byte[] payloadBytes = request.getInputStream().readAllBytes();

// Security: load the secret from configuration — never hard-code it.
// Skipping signature verification accepts forged webhook payloads as legitimate.
String secret = System.getenv("SCALEKIT_WEBHOOK_SECRET");

boolean ok = scalekitClient.webhook().verifyWebhookPayload(
  secret,
  headers,
  payloadBytes
);
if (!ok) {
  // Reject forged or replayed webhooks
  throw new SecurityException("Invalid webhook signature");
}
```

  <CB.ClassMethod name="verifyWebhookPayload">
    Verifies the authenticity and integrity of webhook payloads from Scalekit. This validates the HMAC signature and timestamp (5-minute tolerance window) to ensure the webhook was sent by Scalekit and hasn't been tampered with.

  </CB.ClassMethod>

```java
import java.util.Map;

Map<String, String> headers = new java.util.HashMap<>();
headers.put("webhook-id", "<id>");
headers.put("webhook-timestamp", "<epoch_seconds>");
headers.put("webhook-signature", "v1,<base64sig>");

byte[] payload = requestBodyBytes;

boolean isValid = scalekitClient.webhook().verifyWebhookPayload("whsec_...", headers, payload);
```
