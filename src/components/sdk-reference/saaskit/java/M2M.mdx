  <CB.ClassMethod name="createOrganizationClient">
    Creates a new M2M (machine-to-machine) API client for an organization. The plain secret is returned only at creation time and cannot be retrieved again.

  </CB.ClassMethod>

```java
import com.scalekit.grpc.scalekit.v1.clients.OrganizationClient;

OrganizationClient orgClient = OrganizationClient.newBuilder()
  .setName("Production Service Account")
  .build();

scalekitClient.m2m().createOrganizationClient("org_123", orgClient);
```

  <CB.ClassMethod name="getOrganizationClient">
    Retrieves details of a specific M2M scalekitClient.

  </CB.ClassMethod>

```java
scalekitClient.m2m().getOrganizationClient("org_123", "skc_xxxxx");
```

  <CB.ClassMethod name="updateOrganizationClient">
    Updates the configuration of an existing M2M scalekitClient.

  </CB.ClassMethod>

```java
import com.scalekit.grpc.scalekit.v1.clients.OrganizationClient;

OrganizationClient updates = OrganizationClient.newBuilder()
  .setName("Updated Service Account Name")
  .build();

scalekitClient.m2m().updateOrganizationClient("org_123", "skc_xxxxx", updates);
```

  <CB.ClassMethod name="deleteOrganizationClient">
    Permanently deletes an M2M client from an organization. This operation cannot be undone and all associated secrets are invalidated.

  </CB.ClassMethod>

```java
scalekitClient.m2m().deleteOrganizationClient("org_123", "skc_xxxxx");
```

  <CB.ClassMethod name="createOrganizationClientSecret">
    Creates a new secret for an M2M scalekitClient. The plain secret value is returned only at creation time and cannot be retrieved again.

  </CB.ClassMethod>

```java
scalekitClient.m2m().createOrganizationClientSecret("org_123", "skc_xxxxx");
```

  <CB.ClassMethod name="deleteOrganizationClientSecret">
    Permanently deletes a secret from an M2M scalekitClient.

  </CB.ClassMethod>

```java
scalekitClient.m2m().deleteOrganizationClientSecret("org_123", "skc_xxxxx", "sks_xxxxx");
```
