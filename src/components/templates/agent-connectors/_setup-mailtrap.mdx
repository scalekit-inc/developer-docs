Mailtrap uses a Bearer token for authentication. Generate a token from your Mailtrap account, then register it with Scalekit once per environment — no OAuth app registration required.

<Steps>
1. ### Get your API token

   Log in to [mailtrap.io](https://mailtrap.io) and go to **API Tokens** at [https://mailtrap.io/api-tokens](https://mailtrap.io/api-tokens).

   {/* TODO: add screenshot — alt: "Mailtrap API Tokens page", src: @/assets/docs/agent-connectors/mailtrap/api-tokens.png */}

   Click **Add Token**, enter a name, and click **Create**. Copy the token — it is shown only once.

   {/* TODO: add screenshot — alt: "New API token created in Mailtrap", src: @/assets/docs/agent-connectors/mailtrap/new-token.png */}

   <Aside type="caution" title="Store your token securely">
     The token is shown only at creation time. Save it to a secure secrets manager before closing the dialog.
   </Aside>

2. ### Create a connection

   In the [Scalekit dashboard](https://app.scalekit.com), go to **AgentKit** > **Connections** and click **Create Connection**. Find **Mailtrap** in the list and click **Create**.

   {/* TODO: add screenshot — alt: "Create connection dialog with Mailtrap selected in Scalekit dashboard", src: @/assets/docs/agent-connectors/mailtrap/create-connection.png */}

3. ### Create a connected account

   Go to **Connected Accounts** for your Mailtrap connection and click **Add account**. Fill in:

   - **Your User's ID** — a unique identifier for the user in your system
   - **Bearer Token** — the API token you copied in step 1

   Click **Save**.

   {/* TODO: add screenshot — alt: "Add connected account form for Mailtrap in Scalekit dashboard", src: @/assets/docs/agent-connectors/mailtrap/add-connected-account.png */}
</Steps>
