Register your Tango API key with Scalekit so your agent can search federal contracts, opportunities, and procurement data. Tango uses API key authentication — there is no redirect flow.

<Steps>
1. ### Get a Tango API key

   - Go to the [Tango developer portal](https://tango.io) and sign in.
   - Click **Manage Account** in the top navigation.
   - Find the **API Keys** section. Your **Default Key** is shown there — click **Click to copy** to copy it.
   - To create additional keys, click **Manage API keys →** and then create a new key.

   ![Tango developer portal showing the API Keys section with the Default Key and rate limit information](@/assets/docs/agent-connectors/tangomcp/create-api-key.png)

   <Aside type="note" title="Rate limits">
     The free plan allows 100 requests per day and 25 requests per minute. Monitor your daily remaining requests at the top of the API Keys section.
   </Aside>

2. ### Create a connection in Scalekit

   - In the [Scalekit dashboard](https://app.scalekit.com), go to **AgentKit** > **Connections** > **Create Connection**.
   - Find **Tango MCP** and click **Create**.
   - Note the **Connection name** — use this as `connection_name` in your code (e.g., `tangomcp`).
   - Click **Save**.

3. ### Add a connected account

   Connected accounts link a user in your system to their Tango API key.

   **Via dashboard (for testing)**

   - Open the connection and click the **Connected Accounts** tab → **Add account**.
   - Fill in **Your User's ID** and the **API Key** you copied in step 1.
   - Click **Save**.

   **Via API (for production)**

   <Tabs syncKey="tech-stack">
     <TabItem label="Node.js">
       ```typescript
       // Never hard-code API keys — retrieve from secure storage
       const tangoApiKey = getUserTangoKey(); // retrieve from your secure store

       await scalekit.actions.upsertConnectedAccount({
         connectionName: 'tangomcp',
         identifier: 'user_123',
         credentials: { api_key: tangoApiKey },
       });
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       # Never hard-code API keys — retrieve from secure storage
       tango_api_key = get_user_tango_key()  # retrieve from your secure store

       scalekit_client.actions.upsert_connected_account(
           connection_name="tangomcp",
           identifier="user_123",
           credentials={"api_key": tango_api_key}
       )
       ```
     </TabItem>
   </Tabs>

   <Aside type="tip" title="Production usage">
     Call `upsertConnectedAccount` when a user enters their Tango API key — for example, on a settings page in your app.
   </Aside>
</Steps>
