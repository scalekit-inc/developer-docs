Register your Scalekit environment with the Airtable connector so Scalekit handles the authentication flow and token lifecycle for you. The connection name you create will be used to identify and invoke the connection programmatically. Then complete the configuration in your application as follows:

<Steps>
1. ### Create the Airtable connection in Scalekit

   - In [Scalekit dashboard](https://app.scalekit.com), go to **AgentKit** > **Connections** > **Create Connection**. Search for **Airtable** and click **Create**.

     ![Search for Airtable and create a new connection](@/assets/docs/agent-connectors/airtable/create-airtable-connection.png)

   - In the **Configure Airtable Connection** dialog, copy the **Redirect URI**. You will need this when registering your OAuth integration in Airtable.

     ![Copy the redirect URI from the Configure Airtable Connection dialog](@/assets/docs/agent-connectors/airtable/configure-airtable-connection.png)

2. ### Register an OAuth integration in Airtable

   - Go to the [Airtable Builder Hub](https://airtable.com/create/oauth) and navigate to **OAuth integrations**. Click **Register an OAuth integration**.

     ![OAuth integrations page in Airtable Builder Hub](@/assets/docs/agent-connectors/airtable/airtable-oauth-integrations.png)

   - Fill in your integration details (name, description, and other required fields).

   - Under **OAuth redirect URLs**, paste the redirect URI you copied from the Scalekit dashboard.

3. ### Get your client credentials

   - On your OAuth integration page in the Airtable Builder Hub, find the **Developer details** section.

   - Copy the **Client ID**.

   - Click **Generate client secret** and copy the secret value immediately.

     ![Copy Client ID and generate a client secret from Airtable developer details](@/assets/docs/agent-connectors/airtable/airtable-developer-details.png)

4. ### Add credentials in Scalekit

   - In [Scalekit dashboard](https://app.scalekit.com), go to **AgentKit** > **Connections** and open the Airtable connection you created.

   - Enter your credentials:
     - **Client ID** — from the Airtable developer details
     - **Client Secret** — the generated secret from Airtable
     - **Scopes** — select the permissions your app needs (for example, `data.records:read`, `data.records:write`, `schema.bases:read`, `schema.bases:write`, `webhook.manage`). See [Airtable OAuth scopes reference](https://airtable.com/developers/web/api/scopes) for the full list.

     ![Airtable credentials entered in the Scalekit connection configuration](@/assets/docs/agent-connectors/airtable/airtable-credentials-filled.png)

   - Click **Save**.
</Steps>
