<details>
<summary>Proxy API call</summary>

**Don't worry about your Mailtrap domain in the path.** Scalekit automatically resolves `{{domain}}` from the connected account's configuration. For example, a request with `path="/v2/users/me"` will be sent to `https://mycompany.mailtrap.com/api/v2/users/me` automatically.

<Tabs syncKey="tech-stack">
  <TabItem label="Node.js">
    ```typescript
    const result = await actions.request({
      connectionName: 'mailtrap',
      identifier: 'user_123',
      path: '/v2/users/me',
      method: 'GET',
    });
    console.log(result);
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    result = actions.request(
        connection_name='mailtrap',
        identifier='user_123',
        path="/v2/users/me",
        method="GET"
    )
    print(result)
    ```
  </TabItem>
</Tabs>
