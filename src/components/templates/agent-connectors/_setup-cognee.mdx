Register your Cognee API key with Scalekit so it can authenticate requests on your behalf. You'll need an API key and your Cognee Cloud API host from [platform.cognee.ai](https://platform.cognee.ai).

<Steps>
1. ### Generate an API key

    {/* TODO: add provider-specific steps — confirm the exact navigation path in the Cognee dashboard for creating an API key */}
    - Sign in to [platform.cognee.ai](https://platform.cognee.ai) and go to **Dashboard** → **API Keys**.

    - Click **Create API key**, enter a name (e.g., `Agent Auth`), and confirm.

    - Copy the key and note your API host (e.g., `your-tenant.aws.cognee.ai`) — you'll need both in step 3.

      {/* TODO: add screenshot — alt: "Cognee dashboard API Keys page showing the Create API key button", original src: @/assets/docs/agent-connectors/cognee/create-api-key.png */}

2. ### Create a connection

    In [Scalekit dashboard](https://app.scalekit.com), go to **AgentKit** > **Connections** > **Create Connection**. Find **Cognee** and click **Create**.

      {/* TODO: add screenshot — alt: "Scalekit connection configuration for Cognee showing the API Key authentication type", original src: @/assets/docs/agent-connectors/cognee/create-connection.png */}

3. ### Create a connected account

    Go to **Connected Accounts** for your Cognee connection and click **Add account**. Fill in the required fields:

    - **Your User's ID** — a unique identifier for the user in your system
    - **API Key** — the API key you copied in step 1
    - **API Host** — your Cognee Cloud API host without the `https://` prefix (e.g., `your-tenant.aws.cognee.ai`)

    - Click **Save**.

      {/* TODO: add screenshot — alt: "Add connected account form for Cognee in Scalekit dashboard", original src: @/assets/docs/agent-connectors/cognee/add-connected-account.png */}
</Steps>

<Aside type="note" title="Where to find your API host">
  Your Cognee Cloud API host is shown in the Cognee dashboard alongside your API key. It's tenant-specific, so don't reuse a host value from another Cognee account.
</Aside>
